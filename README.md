✅ What the Project Does

The Student Assignment Deadline Tracker is a simple tool designed to help students stay organized with their academic work. Instead of relying on memory or scattered notes, this system allows users to keep all their assignments in one place.

Students can add details like the assignment title, subject, deadline, and priority. The tracker then displays all tasks in an organized way so it’s easy to see what needs to be done first. It also allows users to update or delete assignments and mark them as completed once finished.

Overall, the project acts like a personal academic assistant that helps students manage their time better, avoid last-minute stress, and stay on top of their deadlines.

✅ How to Run It

Running the project is simple and does not require any complicated setup.

First, download or clone the project files from the repository.
Open the project folder in Visual Studio Code or any code editor.
If it is a web-based project, open the main file (usually index.html) in a browser.

If it is a Python-based program, open the terminal and run the main file using:

python filename.py
Once the program starts, you can begin adding, viewing, and managing assignments.

Make sure all required files are in the same folder so the project runs smoothly without errors.

✅ Example Output

When the program runs, the user is presented with a simple interface where they can manage assignments.

For example:

After adding an assignment like:
Title: Math Homework, Subject: Maths, Deadline: 1 April 2026
The system displays it in a list format such as:
1. Math Homework | Subject: Maths | Deadline: 2026-04-01 | Priority: High | Status: Pending
If the user marks it as completed, it updates to:
1. Math Homework | Subject: Maths | Deadline: 2026-04-01 | Priority: High | Status: Completed
If no assignments are added, the system shows a message like:
No assignments found.

This clear and simple output helps users quickly understand their tasks and track their progress.
